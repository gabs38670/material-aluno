# JSON e localStorage — Aluno

Aula em 7 etapas que sobem de nível. Você escreve tudo no **script.js**. Visual macOS (modo claro). Pré-requisitos: **funções** e **objetos**.

## Como usar
1. Abra o **index.html** no navegador.
2. Aperte **F12** e deixe o **Console** aberto.
3. No **script.js**, cada etapa tem OBJETIVO, DICA e MODELO — a dica quase entrega a resposta. Escreva no espaço marcado, salve e aperte **F5**.

## O caminho (do básico ao avançado)
1. Objeto vira texto (stringify) · 2. Texto vira objeto (parse) · 3. Guardar (setItem) ·
4. Ler (getItem + null) · 5. Objeto salvo (o coração) · 6. Valor padrão (if null) ·
7. Lista salva · Projeto: perfil que lembra de você.

> Você só edita o arquivo **.js**. HTML e CSS já estão prontos.
> Regra de ouro: o localStorage só guarda **texto** — o JSON é o tradutor:
> guardar = `setItem` + `stringify` · ler = `getItem` + `parse`

**Aviso:** abra sempre pelo mesmo navegador — o localStorage é gravado por navegador. Se trocar de máquina ou navegador, os dados salvos não vêm junto.
