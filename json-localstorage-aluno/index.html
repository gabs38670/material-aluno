JSON e localStorage

Como um app lembra de você depois que a página fecha — em sete passos.

1 · Vira texto 2 · Vira objeto 3 · Guardar 4 · Ler 5 · Objeto salvo 6 · Null 7 · Lista Projeto

Até hoje, tudo que você criou morria no F5. Nesta aula isso acaba: o localStorage guarda dados no computador do usuário — mas ele só aceita texto. O JSON é o tradutor que transforma objeto em texto e texto em objeto. Você escreve tudo no script.js. Abra o console com F12.

Etapa 1 · Nível básico

Objeto vira texto — JSON.stringify

JSON é o objeto escrito como texto, no formato que viaja pela internet: é assim que WhatsApp, iFood e o app do busão mandam dados. Repare nas aspas nas etiquetas — é a marca registrada do JSON.

  • stringifyobjeto → texto
  • parsetexto → objeto
  • localStorageonde o texto fica salvo

Etapa 2 · O caminho de volta

Texto vira objeto — JSON.parse

Um texto JSON chegou "da internet". Enquanto for texto, o ponto não funciona. O JSON.parse devolve o objeto de verdade — e aí obj.nome volta a funcionar.

— aguardando o JavaScript —

Etapa 3 · A mágica

Guardar no navegador — setItem

localStorage.setItem("chave", "valor") grava um texto no computador, não na página. Depois de rodar, aperte F5 quantas vezes quiser: o dado continua lá. É a primeira vez que algo seu sobrevive ao refresh.

— aguardando o JavaScript —

Etapa 4 · A armadilha

Ler — getItem (e o null)

getItem devolve o texto guardado — ou null quando a chave não existe, sem dar erro. Lembra do undefined dos objetos? Mesma armadilha, outro nome.

— aguardando o JavaScript —

Etapa 5 · O coração da aula

Guardar um objeto inteiro

O localStorage só aceita texto — então o JSON entra em campo. Decore o par: guardar = setItem + stringify · ler = getItem + parse.

— aguardando o JavaScript —

Etapa 6 · Nível intermediário

E se não tiver nada salvo?

Todo app faz essa pergunta ao abrir: tem algo salvo? Se getItem devolver null, é a primeira visita — use um valor padrão. É um if que você já sabe fazer.

— aguardando o JavaScript —

Etapa 7 · Nível avançado

Lista de objetos salva

O pedido inteiro da lanchonete — uma lista de objetos — também vira texto com o mesmo stringify. Salve, leia de volta e some o total com o for da aula passada. É o esqueleto de um carrinho de compras real.

Projeto final

Perfil que lembra de você

Salve seu nome e curso, aperte F5 e veja a página te reconhecer — igual aos apps que dizem "Bem-vindo de volta". O botão Esquecer apaga tudo (removeItem).
