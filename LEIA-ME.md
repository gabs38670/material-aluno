# Material do Aluno — Lógica e Arrays

## O que tem nesta pasta
- **index.html** — a página (a estrutura). Abra este arquivo no navegador.
- **style.css** — a aparência da página. Você não precisa mexer nele.
- **script.js** — a lógica. **É aqui que você vai escrever**, junto com o professor.

## Como usar
1. Abra o **index.html** no navegador (clique duas vezes).
2. Aperte **F12** e deixe a aba **Console** aberta.
3. A cada etapa, o professor mostra o código. Você digita no **script.js**, salva e aperta **F5** na página.
4. O resultado aparece na caixa preta da etapa correspondente.

## Regra de ouro
Mudou o código e a página não mudou? **Salve o arquivo** e aperte **Ctrl+F5** (atualiza ignorando o cache). É o erro número 1 do primeiro dia.

## Os 6 conceitos da aula
| Conceito | Em uma frase |
|---|---|
| Variável | Caixa com nome para um valor. |
| Array | Uma variável que guarda vários valores. |
| Índice | A posição de um item — **começa no 0**. |
| .length | Quantos itens o array tem. |
| push / pop | Adiciona no fim / remove o último. |
| for | Visita cada posição do array. |
