# Funções — Aluno

Aula sobre **funções** em 7 etapas que sobem de nível. Você escreve tudo no **script.js**. Visual macOS (modo claro).

## Como usar
1. Abra o **index.html** no navegador.
2. Aperte **F12** e deixe o **Console** aberto.
3. No **script.js**, cada etapa tem OBJETIVO, DICA e MODELO. Escreva no espaço marcado, salve e aperte **F5**.

## O caminho (do básico ao avançado)
1. O que é uma função · 2. Parâmetro · 3. Retorno · 4. Reutilização ·
5. Decisão (if) · 6. Composição (função que usa função) · 7. Lista (array) · Projeto: calculadora.

> Você só edita o arquivo **.js**. HTML e CSS já estão prontos.
> Regra de ouro: **criar** uma função não a executa — para rodar, **chame**: `nome();`
