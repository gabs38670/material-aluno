A estrutura é o HTML, a aparência é o CSS — os dois já estão prontos. Quem cria e usa os objetos é o JavaScript, que você escreve no script.js. Abra o console com F12 para acompanhar.

Etapa 1 · Nível básico

O que é um objeto

Pense na sua ficha de matrícula do Senac: nome, curso, unidade — vários dados de uma coisa só, todos juntos. Um objeto é isso: um grupo de dados onde cada um tem uma etiqueta (a propriedade) e um valor.

  • Propriedadea etiqueta do dado
  • Valoro dado em si
  • Pontoobjeto.propriedade lê

Alterar e adicionar

Um celular na vitrine do Largo Treze entra em promoção: o preço muda, mas é o mesmo celular. Com o = você troca o valor de uma propriedade — e também pode criar uma propriedade nova depois do objeto pronto.

Etapa 3 · Objeto + função

O objeto viaja inteiro

Em vez de mandar nome, curso e unidade separados, você manda a ficha inteira para a função — e dentro dela lê o que precisar com o ponto. Um parâmetro só, todos os dados juntos.

Etapa 4 · Saída

Função que devolve com return

Igual à aula de funções: em vez de só mostrar, a função pode devolver um texto montado com os dados do objeto — o crachá pronto para usar onde você quiser.

Decisão com os dados do objeto

No busão do Terminal Santo Amaro, estudante paga meia. A função olha o objeto e decide com if / else: a resposta depende dos dados de quem chegou.

Etapa 6 · Fábrica

Função que fabrica objetos

A barraca da feira tem dezenas de produtos. Escrever um objeto na mão para cada um? Não — uma função recebe os dados e devolve o objeto pronto. É função + return + objeto, tudo junto.

Etapa 7 · Nível avançado

Lista de objetos

O pedido da lanchonete é uma lista de objetos: cada item tem nome e preço. Percorra a lista com for e some os preços — é assim que um carrinho de compras calcula o total.

O vocabulário dos objetos

TermoEm uma frase
ObjetoGrupo de dados que andam juntos, entre { }.
PropriedadeA etiqueta de um dado, ex: nome, preco.
ValorO dado em si, ex: "Maria", 8.
PontoLê ou troca um dado: objeto.propriedade.
FábricaFunção que recebe dados e devolve (return) um objeto pronto.
MétodoFunção guardada dentro do objeto — console.log é um!
