# Objetos — Aluno

Aula sobre **objetos** em 7 etapas que sobem de nível. Você escreve tudo no **script.js**. Visual macOS (modo claro). Pré-requisito: a aula de **funções**.

## Como usar
1. Abra o **index.html** no navegador.
2. Aperte **F12** e deixe o **Console** aberto.
3. No **script.js**, cada etapa tem OBJETIVO, DICA e MODELO — a dica quase entrega a resposta, leia com calma. Escreva no espaço marcado, salve e aperte **F5**.

## O caminho (do básico ao avançado)
1. O que é um objeto · 2. Alterar e adicionar · 3. Objeto + função · 4. Return com objeto ·
5. Decisão (if) · 6. Fábrica de objetos · 7. Lista de objetos · Projeto: crachá do curso.

> Você só edita o arquivo **.js**. HTML e CSS já estão prontos.
> Regra de ouro: um objeto **agrupa dados que andam juntos** — para ler um dado: `objeto.propriedade`
